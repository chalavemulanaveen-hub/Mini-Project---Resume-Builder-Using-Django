# Resume Builder using Python and Django

Resume Builder is a web application developed using Python and Django that allows users to create professional resumes quickly and easily. Users can choose from multiple resume templates, enter their personal, educational, and professional details, preview the generated resume, and download it as a PDF.

## Features

* Multiple Resume Templates
* User-Friendly Interface
* Real-Time Resume Preview
* PDF Download Functionality
* Responsive Design
* Django Backend Integration
* Resume Data Storage using SQLite
* Easy Template Customization

## Technologies Used

* Python
* Django
* HTML5
* CSS3
* JavaScript
* SQLite
* HTML2Canvas
* jsPDF

## Project Modules

* Template Selection
